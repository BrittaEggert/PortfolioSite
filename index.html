My work

PROgress

PROgress is a web application especially made for women to get advice from experienced women within the fitness industry.

My role:

I was responsible for conducting user tests, creating personas, user flows, user journeys,
setting up a navigation plan, creating wireframes, the logo design, writing down the UI Design Style guide,
polishing up final mockups, and creating a clickable prototype.

Used tools:

Tools I used for the project were: Balsamiq, Adobe XD, Miro, Figma, Flowmapp, Typeform,
Usability hub, Illustrator, Photoshop, Post-its, pen, and paper.
